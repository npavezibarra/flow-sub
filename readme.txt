A custom WordPress plugin for subscription management.

== Description ==

Flow Sub is a lightweight plugin designed to handle custom subscription flows.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/flow-sub` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
